빵잇나우의 컬러는 ‘따뜻하게 구워진 빵’의 포근함을 표현하는 **오렌지(Primary)**와   신선하고 생동감 있는 **민트(Secondary)**를 중심으로 구성했습니다.

| 역할      | 컬러 코드 | 의미                               |
| --------- | --------- | ---------------------------------- |
| Primary   | `#FF7651` | 따뜻하고 포근한 빵의 느낌          |
| Secondary | `#26CF9F` | 신선함과 생동감을 더하는 민트 컬러 |
